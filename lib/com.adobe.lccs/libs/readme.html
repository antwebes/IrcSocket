This directory contains three SWCs that support that latest released features:
  • player9 provides a lccs.swc and lccs source files that supports Flash player 9.
  • player10 provides a lccs.swc and lccs source files that supports Flash player 10.
  • player10.1 provides a lccs.swc that supports Flash player 10.1.
  • flashOnly provides lccs.swc and lccs source for Flash only SDK for Flash Player 9, 10 and 10.1.
