Class SharedCursorPane

SharedCursor tracks the cursor positions of any user with role of UserRoles.PUBLISHER or higher within it and reports that position to other users. It is also responsible for rendering the remote cursors corresponding to these positions.

EventSummary
synchronizationChange Dispatched when the SimpleChatModel has fully connected and synchronized with the service or when it loses that connection.

The field of UserDescriptor you want to show as label with your cursor. You can also show custom fields
nodeName
When specifying an existing CollectionNode, nodeName specifies a nodeName to use within that collectionNode for all message traffic. Defaults to "Shared_Cursors".
pollInterval
The time in milliseconds of the polling interval for cursor positions. Note that setting this to a higher number reduces network message traffic but also reduces responsiveness. Setting it to a lower number uses more bandwidth.
sharedID
Defines the logical location of the component on the service; typically this assigns the sharedID of the collectionNode used by the component. sharedIDs should be unique within a room if they're expressing two unique locations. Note that this can only be assigned once before subscribe() is called. For components with an id property, sharedID defaults to that value.
sizingMode
Property for sharing mode. There are two modes of sharing, absolute and relative. In absolute mode you get the exact position of the cursor of the sharing user. In relative you get the relative position with respect to the size of your own cursor pane. Default mode is absolute.
width
Number that specifies the width of the component, in pixels, in the parent's coordinates.

Number that specifies the component's horizontal position, in pixels, within its parent container.

Number that specifies the component's vertical position, in pixels, within its parent container.
