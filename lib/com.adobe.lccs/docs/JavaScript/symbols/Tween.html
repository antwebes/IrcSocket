Class Tween

The Tween class defines a tween, a property animation performed on a target object over a period of time. When the constructor is called, the animation automatically starts playing.
Defined in: LCCS_JS_Bridge.js.

Class Detail
Tween(listener, init, end, dur)
Parameters:
listener
Object that is notified at each interval of the animation. You typically pass the this keyword as the value. The listenerObj must define the onTweenUpdate() method and optionally the onTweenEnd() method. The former method is invoked for each interval of the animation, and the latter is invoked just after the animation finishes.
init
Initial value(s) of the animation. Either a number or an array of numbers. If a number is passed, the Tween interpolates between this number and the number passed in the endValue parameter. If an array of numbers is passed, each number in the array is interpolated.
end
Final value(s) of the animation. The type of this argument must match the startValue parameter.
dur
Duration of the animation, expressed in milliseconds.
