Packagecom.adobe.rtc.session.sessionClasses
Classpublic class PlaybackSeekClient

This class is a NetStream "client" used by SessionManagerPlayback while seeking. It stores the state for a collection until the seek point has been reached and then call the onSeekComplete callback.
